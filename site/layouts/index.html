clnkr

EVALUATIONS clankerval(1)

NAME

clankerval is a standalone evaluation runner for clnkr and clnku. It lives in its own repository and is installed separately.

INSTALL

./scripts/install-clankerval.sh

DESCRIPTION

clankerval owns eval execution, result reports, and raw bundles. The checked-in evaluations/ directory in this repo defines the suites; clankerval runs them. clnkeval is a compatibility alias to the same binary.
