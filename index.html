NAME clnkr(1)

NAME

clnkr

A minimal, stubborn coding agent with a rough metal finish.

SYNOPSIS

  • Homebrew: brew install clnkr-ai/tap/clnkr
  • Debian/Ubuntu: sudo apt install ./clnkr_1.5.4-1_amd64.deb

DESCRIPTION

clnkr works with a human in the loop, or hand it the wheel with --full-send. Permissions are simple, sharp, and not here to save you from yourself.

EVALUATIONS clankerval(1)

NAME

clankerval is a standalone evaluation runner for clnkr and clnku. It lives in its own repository and is installed separately.

INSTALL

./scripts/install-clankerval.sh

DESCRIPTION

clankerval owns eval execution, result reports, and raw bundles. The checked-in evaluations/ directory in this repo defines the suites; clankerval runs them.
