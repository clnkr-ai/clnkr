NAME clankerval(1)

NAME

evaluations

A minimal evaluation runner for clnkr and clnku.

SYNOPSIS

  • Debian-family example: sudo apt install ./clankerval_<version>-1_<arch>.deb

DESCRIPTION

clankerval owns eval execution, result reports, and raw bundles. The checked-in evaluations/ directory in this repo defines the suites; clankerval runs them.
